
# Java Console Calculator

A simple Java console-based calculator that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Loop until user exits
- Handles divide-by-zero error

## Technologies Used
- Java (JDK)
- Scanner Class
- VS Code / IntelliJ

## How to Run

1. Compile:
   javac src/ConsoleCalculator.java

2. Run:
   java -cp src ConsoleCalculator

## Learning Outcome
- Understanding Java syntax
- Methods and control flow
- User input handling
- Switch statements
